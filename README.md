# ModernFit
Im Moduel WECOTE - Web- & Communication Technologies dürfen wir als Endabgabe eine eigene Web App bauen. Wir haben uns dazu entschieden, dass wir eine App bauen, welche Fitnessgängern hilft, ihr Köprergewicht zu tracken. Sie könnn so einfach den Fortschritt im Auge behalten und werden motiviert, wenn ein Gewichtsverlust eintritt!
